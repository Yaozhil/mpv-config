# 独特功能说明

此文件记录本整合包区别于 `dyphire/mpv-config` 原始整合思路的主要变化，方便后续宣传、维护和同步上游时保留自己的特色。

## 1. uosc 底栏重新组织

底栏是这一版最显眼的变化。常用入口被集中到底部一层，减少反复进入多级菜单：

- 文件打开、历史、目录入口保留在底栏左侧。
- 弹幕入口单独放到底栏，方便追剧时随时开关和匹配。
- 字幕、音轨、章节、倍速、播放列表、统计信息和更多菜单都放在可见区域。
- 播放态与空闲态分别优化：播放时突出播放控制，空闲时突出打开文件和网盘入口。

## 2. 中文弹幕体验

整合包面向中文追剧用户，弹幕相关入口更靠前：

- 支持网络弹幕搜索、匹配和加载。
- 支持为 URL 串流场景自动加载弹幕。
- 弹幕提示位置经过调整，尽量避开播放器顶部其他信息。
- 弹幕性能设置对 4K/高刷新场景做了限制，避免同屏弹幕过多拖慢 OSD 渲染。

## 3. 片头片尾标记与自动跳过

这一版把片头片尾作为底栏一等入口，而不是隐藏在深层脚本功能里：

- 支持 Anime Skip / TheIntroDB 片段查询。
- 支持本地手动标记片头片尾。
- 支持网站片段的本地修正覆盖。
- 支持把片段显示到时间轴，并配合自动跳过。
- 手动模板可复用，适合电视剧/动画连续剧场景。

## 4. 中文化 stats 信息

统计信息面板更适合中文用户阅读：

- 文件、显示器、视频、音频、解码、HDR、帧耗时等信息中文化。
- 便于快速判断硬解、刷新率、掉帧、色彩空间和码率。
- 配合底栏媒体标签，可在播放时快速了解当前播放状态。

## 5. AList/OpenList 网盘入口

空闲底栏新增网盘按钮，并把配置表单做进播放器 UI：

- 地址、账号、密码在 uosc 菜单中行内编辑。
- 保存后可直接打开 AList/OpenList 文件浏览。
- 播放 AList 文件时可回到当前文件所在目录。
- 发布版不会携带个人网盘地址、账号和密码。

当前实现基于 AList/OpenList API，不等同于通用 WebDAV PROPFIND 客户端。

## 6. 本地 ISO 播放链路

本地 ISO 播放入口做了统一处理：

- 拖拽、`Ctrl+O`、file-browser、最近播放等入口都尽量传入真实 `.iso` 路径。
- 由统一脚本改写为 `bd://` / `dvd://` 播放。
- 最近播放和历史记录尽量还原真实 ISO 路径，而不是只记录 `bd://`。

注意：AList/OpenList 远程 ISO 不作为当前发布功能，建议下载到本地后播放。

## 7. 公开发布清理

公开版应避免携带个人运行痕迹：

- 不带播放历史、watch_later、日志、截图和 shader cache。
- 不带个人 API key、token、Trakt secret、网盘账号密码。
- 不绑定固定显卡、固定音频设备或本机绝对路径。
- 字体、二进制播放器和大型资源建议通过 Release/网盘分发，仓库只保留说明和必要配置。
